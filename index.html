DarkPatterns-LLM

A Multi-Layer Benchmark for Detecting Manipulative and Harmful AI Behavior

Sadia Asif 路 Israel A. Rosales Laguan 路 Haris Khan 路 Shumaila Asif 路 Muneeb Asif

Paper arXiv Code Dataset

Abstract

Large Language Models exhibit subtle manipulative behaviors that evade existing safety benchmarks. We introduce DarkPatterns-LLM, a multi-layer benchmark that evaluates manipulation across seven harm categories using a four-stage analytical pipeline.

Key Contributions

  • First multi-dimensional benchmark for manipulation detection in LLMs
  • Seven-category harm taxonomy grounded in psychology and ethics
  • Four-layer evaluation pipeline with interpretable diagnostics
  • Novel metrics: MRI, CRS, SIAS, THDS

Framework Overview

Multi-layer pipeline: detection, stakeholder analysis, temporal modeling, and risk alignment.

Dataset

The dataset contains 401 expert-annotated examples spanning seven harm categories: Legal/Power, Psychological, Emotional, Physical, Autonomy, Economic, and Societal harm.

Results Highlights

  • Claude 3.5 achieves highest MRI (89.7)
  • All models struggle with Autonomy Harm detection
  • Temporal harm modeling remains a major open challenge
